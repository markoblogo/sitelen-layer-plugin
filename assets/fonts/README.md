# Bundled sitelen pona font

This package bundles `sitelen seli kiwen asuki` as the default ligature-capable sitelen pona font for `sitelenPona.renderStrategy: 'ligature-font'`.

- Font: sitelen seli kiwen asuki
- Author: KreativeKorp / jan Lepeka
- Source: https://github.com/kreativekorp/sitelen-seli-kiwen
- Website: https://www.kreativekorp.com/software/fonts/sitelenselikiwen/
- Bundled file: `sitelen-seli-kiwen-asuki.ttf`
- Source version: Git tag `2.2`
- License: SIL Open Font License 1.1

The upstream documentation states that the `asuki` version automatically converts toki pona written in Latin letters into sitelen pona using font ligatures. The font is bundled under the SIL Open Font License; see `OFL-sitelen-seli-kiwen.txt` in this directory.
