# v0.3.0 Integration Track (PoC)

Набор стартовых шаблонов для roadmap-итерации `v0.3.x`.

## A) Browser extension PoC (Manifest V3)

`manifest.json` (минимум):

```json
{
  "manifest_version": 3,
  "name": "Sitelen Layer Reader",
  "description": "Optional read-only sitelen-layer toggle for toki pona pages.",
  "version": "0.0.1",
  "permissions": ["storage", "scripting", "activeTab", "tabs"],
  "host_permissions": ["http://*/*", "https://*/*"],
  "background": {
    "service_worker": "background.js",
    "type": "module"
  },
  "action": {
    "default_title": "Sitelen layer"
  },
  "content_scripts": [
    {
      "matches": ["<all_urls>"],
      "js": ["content.js"],
      "run_at": "document_idle"
    }
  ]
}
```

`content.js` (injection, allowlist + toggle):

```js
const ALLOW = [/toki-pona|\.tp\//i];

function isAllowedUrl(url) {
  return ALLOW.some((rule) => rule.test(url));
}

if (isAllowedUrl(location.href)) {
  const mount = '#sitelen-layer-toggle';
  const script = document.createElement('script');
  script.type = 'module';
  script.textContent = `
    import { createSitelenLayerPlugin } from 'sitelen-layer-plugin';
    createSitelenLayerPlugin({
      container: 'body',
      toggleMount: '${mount}',
      toggleMode: 'auto',
      sitelenPona: { renderStrategy: 'ligature-font' }
    }).init();
  `;
  document.documentElement.appendChild(script);
  script.remove();
}
```

`background.js` (site allowlist + action toggle, skeleton):

```js
chrome.runtime.onInstalled.addListener(() => {
  chrome.storage.local.set({ enabled: true });
});
```

> В релизе нужен полноценный CDN/IIFE or bundled asset + safe CSP strategy and user options UI; здесь — только PoC-скелет.

## B) CMS интеграторы

### WordPress

Пример `enqueue` в теме/плагине:

```php
function tp_layer_assets() {
  wp_register_style('sitelen-layer', 'https://cdn.jsdelivr.net/npm/sitelen-layer-plugin/dist/sitelen-layer-plugin.css', [], '0.0.0');
  wp_register_script('sitelen-layer', 'https://cdn.jsdelivr.net/npm/sitelen-layer-plugin/dist/sitelen-layer-plugin.iife.js', [], '0.0.0', true);

  wp_add_inline_script('sitelen-layer', "\
    window.addEventListener('DOMContentLoaded', function() {\
      if (typeof window.createSitelenLayerPlugin === 'function') {\
        window.createSitelenLayerPlugin({\
          container: '#content',\
          toggleMode: 'auto',\
          defaultLayer: 'latin'\
        }).init();\
      }\
    });\
  ", 'after');

  wp_enqueue_style('sitelen-layer');
  wp_enqueue_script('sitelen-layer');
}
add_action('wp_enqueue_scripts', 'tp_layer_assets');
```

### Obsidian (plugin PoC)

```ts
import { Plugin } from 'obsidian';
import { createSitelenLayerPlugin } from 'sitelen-layer-plugin';

export default class SitelenLayerObsidianPlugin extends Plugin {
  private pluginInstance: ReturnType<typeof createSitelenLayerPlugin> | null = null;

  onload() {
    this.pluginInstance = createSitelenLayerPlugin({
      container: '.markdown-preview-view',
      toggleMode: 'inline',
      defaultLayer: 'latin'
    });
    this.pluginInstance.init();
  }

  onunload() {
    this.pluginInstance?.destroy();
    this.pluginInstance = null;
  }
}
```

### Logseq (plugin seed)

```ts
import { logseq } from '@logseq/libs';
import { createSitelenLayerPlugin } from 'sitelen-layer-plugin';

let plugin: ReturnType<typeof createSitelenLayerPlugin> | null = null;

export const main = async () => {
  logseq.App.onRouteChanged(async () => {
    plugin?.destroy();
    plugin = createSitelenLayerPlugin({
      container: '.markdown-body',
      toggleSize: 'md',
      showToggle: false
    });
    plugin.init();
  });
};
```

## C) Что считать готовым в релизе v0.3.x

- Единый модульный wrapper/пакет под каждую CMS, не копипаста в нескольких проектах.
- Конфигурируемый allowlist + безопасная модель опций.
- Отдельный pipeline для обновления маппинга и аналитики.
